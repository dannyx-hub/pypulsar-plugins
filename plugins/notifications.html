Notifications Plugin

Native system notifications for PyPulsar apps

pypulsar plugin install pypulsar.notifications

Description

Add beautiful native system notifications to your PyPulsar application. Works on Windows (toast), macOS (Notification Center), and Linux (libnotify).

Uses plyer on Windows/Linux and pync on macOS – zero configuration needed.

Usage

After installation, the plugin adds engine.notifications(title, message) method:

# In any hook or event handler
def on_task_added(event_name, data):
    if hasattr(engine, "notifications"):
        engine.notifications(
            title="New Task",
            message=f'Added: "{data["text"]}"'
        )
